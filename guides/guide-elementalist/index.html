元素组合#

  • 火 + 风 = 火焰风暴（范围+50%）
  • 冰 + 雷 = 冰冻雷（附带冰冻）
  • 风 + 冰 = 寒冰刃（附带减速）
  • 雷 + 火 = 爆裂（暴击率翻倍）

PVE爆发循环#

风刃（聚怪）→ 火球术 → 烈焰风暴 → 闪电链 → 雷暴
